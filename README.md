# 公基刷题

公共基础知识在线刷题应用，收录浙江省事业单位考试真题。

## 题库

- **621 道真题** 来自 2012-2021 年浙江省事业单位考试
- 单选题 407 道 | 多选题 71 道 | 判断题 143 道
- 覆盖政治、法律、经济、管理、公文、地理、科技、历史、文化、时政等分类

## 功能

- 分类筛选 + 题型筛选（单选/多选/判断）
- 练习模式 + 错题复习
- 答题统计，进度追踪
- 数据存 localStorage，刷新不丢失
- 单文件部署，无需服务器

## 部署

直接打开 `index.html` 即可使用，或部署到任意静态托管服务（GitHub Pages、Netlify 等）。
