# 公基刷题

公共基础知识在线刷题工具，题库来源为浙江省宁波市事业单位历年真题。

## 功能

- **621道真题**：单选题 407 道、多选题 71 道、判断题 143 道
- **分类筛选**：按政治、法律、经济、文史等类别筛选
- **题型过滤**：单选 / 多选 / 判断 三种题型切换
- **练习模式**：随机出题，答完显示正确答案和来源
- **错题模式**：收录答错的题目，集中复习
- **多身份切换**：支持多人在同一浏览器分开记录，点击顶部身份标签切换，数据互不干扰
- **答题统计**：总答题数、正确率、各分类进度一目了然
- **清空错题本**：一键清空错题，保留答题记录
- **重置数据**：完全清空所有答题记录

## 技术栈

单文件 HTML（CSS + JS），无需构建、无需后端，开箱即用。
